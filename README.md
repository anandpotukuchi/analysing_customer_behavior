# Customer Behavior Data Analytics Project

An end-to-end **Data Analytics portfolio project** demonstrating how raw customer transaction data is transformed into **actionable business insights**. The project mirrors real-world analytics workflows and highlights proficiency across data preparation, SQL-based analysis, and executive-ready dashboards.

---

## 📊 Business Problem

Businesses often collect large volumes of customer data but struggle to extract insights that drive revenue and retention.  
This project analyzes customer shopping behavior to help stakeholders:
- Understand **who their customers are**
- Identify **what drives purchases and revenue**
- Improve **decision-making using data-backed insights**

---

## 🎯 Key Objectives

- Segment customers based on purchasing behavior  
- Identify purchase trends and seasonal patterns  
- Analyze revenue contribution and key performance indicators (KPIs)  
- Translate analytical findings into business recommendations  

---

## 🔍 Analytical Workflow

### 1. Data Preparation & EDA (Python)
- Cleaned and transformed raw customer data  
- Handled missing values and inconsistencies  
- Conducted exploratory data analysis using **Pandas** and **NumPy**

### 2. SQL-Based Analysis (MySQL)
- Designed relational database schema  
- Loaded processed data into MySQL  
- Wrote optimized SQL queries to answer business-driven questions  

### 3. Data Visualization (Power BI)
- Connected Power BI directly to MySQL  
- Built interactive dashboards to track:
  - Customer segments  
  - Revenue trends  
  - Purchase frequency and value  

### 4. Insights & Reporting
- Synthesized findings into clear insights  
- Delivered data-driven recommendations aligned with business goals
